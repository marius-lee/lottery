## 判定

**通用方法汇编，无独有算法。** 唯一看似独特的"号码趋势6形态"本质是李相春三浪的延伸命名。"K线图法"即MA移动均线。"核心数字组合法"是预计算投注代码表（手工查表用）。

- 算法价值: 0
- 实现优先级: 永不
