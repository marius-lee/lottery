## 一、核心哲学: 均衡原理

全书底层思想: 彩票号码的运行受 **均衡原理** 支配 — 任何指标偏离理论概率后，在"求均衡"作用下会**调偏回补**。

| 概念 | 含义 |
|------|------|
| 均衡原理 | 指标的长期实际出现频率必然趋向理论概率 |
| 求均衡 | 偏离后向均值回归的驱动力 |
| 调偏回补 | 低于理论值的指标会在后续期数中超额出现以"补回" |
| 反转率 | 指标连续遗漏 N 期后下一期出现的概率 |

举例: 奇形态理论概率 50%，若 13 期内只出现 2 次 (15.4%)，则在求均衡作用下判断接下来高概率回补。

## 二、八类蓝球走势图 (八大指标维度)

1. **蓝球号码走势图** — 原始号码序列, 观察落点区域、遗漏
2. **蓝球号码振幅走势图** — 相邻期差值, 观察波动幅度
3. **蓝球尾数走势图** — 号码尾数 0-9, 缩小备选范围
4. **蓝球尾数振幅走势图** — 尾数变化幅度
5. **蓝球两码合走势图** — 两码之和
6. **蓝球两码差走势图** — 两码之差
7. **蓝球两码组合走势图** — 号码对共现规律
8. **蓝球行列分区走势图** — 4×4 矩阵的行/列分区

## 三、实战方法论: 三观五看 + 交叉应用

### 三观 (三级时间尺度观察)
- **观大势**: 长期 (100+ 期) 观察指标总体冷热趋势
- **观阶段**: 中期 (20-50 期) 观察阶段内偏倚状态
- **观局部**: 短期 (5-10 期) 寻找具体图形特征

### 五看 (五种图形特征)
- 竹节形 (递增/递减序列)
- 斜连形 (对角线走势)
- WM 形 (规则 M/W 波动)
- 平台形 (连续多期同值)
- 孤点形 (孤立异常点)

### 指标选用原则
- **宁精勿滥**: 只用"明显态势"的指标, 模糊的不如不用
- **交叉应用**: 至少两个独立走势图交叉验证, 取交集
- **主次分明**: 一行选 + 一列选 = 交集缩小到 1-3 个号码

### 操作流程
```
1. 观察蓝球常规走势图 → 找可用的"明显态势"指标 → 得出备选范围 A
2. 观察蓝球形态走势图 (奇偶/大小/质合/012路) → 交叉验证 → 得范围 B
3. 观察蓝球分区走势图 (行列/分区) → 找"重中之重"号码 → 得范围 C
4. A ∩ B = 交集备选 → 再与 C 对比 → 确定 1-3 个最终号码
```

## 四、三个公式选蓝法 (Chapter 6)

### 方法 1: 五期断蓝法

**原理**: 近 5 期蓝球均值 ±4 作为当期蓝球范围。

**算法**:
```
avg = round(mean(last_5_blues))
low = avg - 4
high = avg + 4
candidates = [n for n in range(1, 17) if low <= n <= high]
```

**效果**: 将 16 选 1 缩小到约 8-9 选 1。书中声称准确率较高, 即使出错也极少连续多期错误。

**可集成性**: ⭐⭐⭐ 极其简单, 5 行代码, 可立即加入 `micro_portfolio.py` 的 `_blue_freq_weights()` 中, 用五期均值附近的蓝球提高权重。

### 方法 2: 矩阵杀蓝法

**原理**: 将 16 个蓝球按 mod 4 分四区, 放入 2×8 矩阵。从前 2 期蓝球做"螺旋计数", 数到的区直接排除 4 个号。

**矩阵结构**:
```
区1: 01,05 | 区2: 02,06 | 区3: 03,07 | 区4: 04,08
     09,13 |      10,14 |      11,15 |      12,16
```

**顺时针杀法**: 从 prev_2 的蓝球开始, 数到 prev_1, 步数 = n。再从 prev_1 数 n 步, 落点所在区排除 4 个。

**逆时针杀法**: 同上但方向相反。

**特殊规则 (绝杀)**: 连续 3 期蓝球落同一区 → 该区 4 个全部排除 (历史上极少连续 4 期同区)。

**错误处理**: 连续 2 期同区是主要错误来源 (209 期中有 64 期)。遇到此情况→逆向思维, 保留该区号码。

**效果**: 通常杀 4-8 个, 准确率约 80%。遇到连续 2 期同区时反转使用。

**可集成性**: ⭐⭐ 作为硬过滤规则之一 (类似现有 `_check_hard`), 排除低概率区的蓝球。

### 方法 3: 32 种公式杀蓝法

32 个简单算术公式, 每个排除 1-3 个蓝球尾数。前 2 个示例:

**(1) 上期蓝球 +6 → 排除尾数**
```
tail = (prev_blue + 6) % 16  # >16 则减 16 再取尾数
exclude_numbers_with_tail(tail)
```

**(2) 上期蓝球 ±10 → 排除两个尾数**
```
t1 = (prev_blue + 10) % 16
t2 = abs(prev_blue - 10) % 16
exclude_numbers_with_tails(t1, t2)
```

**可集成性**: ⭐ 可作软参考, 但不建议硬排除 (单个公式准确率有限)。

## 五、与现有系统的结合点

| 书的思路 | 对应现有模块 | 可做的改进 |

## 六、实操建议

**最小可行集成** (10 行代码):
在 `_blue_freq_weights()` 中, 对五期均值 ±4 范围内的蓝球乘以 1.5 权重, 实现"五期断蓝法"的算法化。

```python
# 五期断蓝法: 近5期均值附近的蓝球加权
recent_5 = [r[7] for r in data[-5:]]
avg = round(sum(recent_5) / 5)
for n in range(max(1, avg - 4), min(16, avg + 4) + 1):
    weights[n - 1] *= 1.5  # 均值±4范围内加权
```
