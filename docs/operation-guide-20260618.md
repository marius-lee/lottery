## 模式详细说明

### 智能均衡 (默认推荐)
红球：贪心 max-min Jaccard 最大化注间距离  
蓝球：Laplace 平滑频率 × 五期断蓝法（近5期均值±4范围加权1.5倍）  
效果：**注间不重叠 + 蓝球短期动量捕获**

### 积极分散
红球：贪心 max-min Jaccard（从1000候选池最优分散选3注）  
蓝球：Laplace 平滑频率  
效果：**纯分散最大化，不引入短期偏置**

### 热度追踪
红球：近10期各位置出现频率加权  
蓝球：Laplace平滑 × 五期断蓝法  
效果：**锁定近期热号趋势**

### 冷号反弹
红球：去连号/大间距/新位置号 + 注间≤2重叠约束  
蓝球：Laplace平滑频率  
效果：**排除历史罕见模式，守冷号回补**

### 纯随机基线
红球：仅硬过滤（去等差序列 + 去历史已开）  
蓝球：纯 Laplace 平滑频率  
效果：**无任何算法增强，用于A/B对比**

## 日常推荐操作

1. 打开页面 → 默认「智能均衡」已选中
2. 点「生成号码」→ 3 注最优分散
3. 点「确定保存」→ 记录到数据库
4. 开奖后 → 「开奖对比」面板 → 「对比最新开奖」

## 如何对比策略效果

1. 选「纯随机基线」→ 生成 → 保存
2. 选「智能均衡」→ 生成 → 保存
3. 开奖后用「开奖对比」面板看各策略命中率
4. 连续 20 期后看哪个策略红球命中最高
