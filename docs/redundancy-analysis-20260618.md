# 算法策略冗余分析 — 2026-06-18

## 评估维度

- 数学基础：是否有恒等式/概率论支撑
- 可验证性：是否有测试/可回测对比
- 独立性：是否与其他策略功能重叠
- 操作负担：用户使用复杂度

## 逐项评估

### 保留项

| 策略 | 理由 |
|------|------|
| 贪心优化 | Jaccard 距离最大化，min(贪心) ≥ min(随机)，数学可证 |
| 回测排名 | 唯一的历史回测选注路径，不与任何策略重叠 |
| 覆盖设计 | Steiner t-wise，独立于采样路径 |
| 幸运开奖(pure) | 纯位置加权独立抽号，独立路径 |

### 合并项

| 合并前 | 合并后 | 理由 |
|--------|--------|------|
| 位置软过滤 + 奇偶/和值 | 高级过滤 | 同属"排除极端组合"，功能同类 |
| 五期断蓝 + 图形规律 | 蓝球智能加权 | 同属蓝球权重调节，叠在一起效果共同 |

### 删除项

| 策略 | 冗余原因 |
|------|----------|
| 运气规则(blend) | 67%接受率几乎无偏置效果；与贪心优化冲突（贪心选出分散组合→运气随机拒绝→破坏分散）；pure模式已独立为幸运开奖按钮 |
| 捆绑投注 | Lift≠未来共现概率；操作链太长（6步）；强制插入破坏贪心分散；降级逻辑不可预测 |

### 保留（降权）

| 策略 | 说明 |
|------|------|
| 分散红球 | 贪心已开启时被贪心覆盖，但贪心关闭时可独立使用 |
