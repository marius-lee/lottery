## 可实现的算法 (2个)

### 1. 十二值选号法 (Ch6§3, p228-236) 【推荐实现】

**原理**: 用约12个位置差值对应的号码作为选号候选。差值公式固定, 每期号码不同但属性相同。

**效果**: 
- 21期测试: 47.62%的期数从~12个号码中开出4+个红球
- 61.9%的期数开出3+个
- 95.24%的期数开出2+个
- 理论期望: ~12个号码应开出2.18个, 实际开出3.10个(65/21)

**关键差值公式**(从已提取的统计表):
| 差值 | 最频值 | 频率 |
|------|--------|------|
| 第5-第3个 | 7,8 | 各8.77% |
| 第6-第3个 | 11 | 7.35% |
| 第5-第4个 | 1 | 17.08%(连号!) |
| 第6-第5个 | 1 | 20.19%(连号!) |
| ... | | |

**实现难度**: 低 — 纯公式计算, 不需要依赖杀号方法库

### 2. 八值选号法 (Ch8§1, p294-300) 【推荐实现】

**原理**: 用胜率最低的11种后区杀号方法反向定蓝球候选。平均每期约7.8个蓝球候选。

**效果**:
- 948/1766次成功 = 53.68%
- 理论值: 7.779/16 = 48.62%
- 超过理论值5个百分点

**11种杀号方法**(胜率从低到高):
1. U-22: 后区号码-22 (92.87%)
2. V4: 前区第4个-后区 (92.87%)
3. Ms2: 前区第5个-第2个
4. 0s: 前区第5个+1
5. 0+2: 前区第4个-2
6. U+24: 后区+24
7. 032: 前区第3个+2
8. P4: 上两期前区第4个互减
9. U: 后区+7
10. U-2: 后区-25
11. Ux: 后区+4

**连续出错规律** (用于决定何时使用):
- 只错1次 → 下期正确: 52.01%
- 连续错2次: 22.93%
- 连续错3次: 13.24%
- 连续错4次: 6.38%
- 连续错5次+: 5.44%

**实现难度**: 中 — 需要实现11个杀号子方法, 每个方法需处理模16/模33 wrap-around

## 不推荐实现的

### 定1个胆码 (Ch5§2, p129-135)
- 依赖18种前区杀号方法, 这些方法散布在Ch3的590种中
- 虽然效果不错(97.45%成功率, ~12.4候选), 但需要完整的杀号方法库
- **太重, 不建议**

### 轮流杀号法 (Ch3§6, Ch4§3)  
- 前区: 590种杀号方法中选top-N轮流 → 最高86.30%
- 后区: 177种方法中选top-N轮流 → 最高95.98% (W4)
- **实现590+177个杀号方法不现实, 放弃**

### 定2胆/定3胆/伴生现象
- 纯统计排名, 不是算法。可作为参考数据但不值得编码
