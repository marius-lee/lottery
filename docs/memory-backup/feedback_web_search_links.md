🚨 铁律：先想再搜，不是先搜再想。

**Why:** 对于基础领域知识（量化方法论、Python标准库、常见算法），直接用自己的训练数据回答。只有在以下情况才搜索：
1. 验证时效性（2024-2025 的最新发展、库版本更新）
2. 自己不确定的具体知识点
3. 用户明确要求搜索

**禁止:** 不加思考就搜、已知概念也搜、用搜索替代自己的判断。

对于基本量化概念（Walk-Forward、IC、夏普、回测方法论等），不需要搜索 — 直接基于训练知识回答。

搜索后必须附来源链接。
