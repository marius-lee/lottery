# 黄贞秀《胆拖拖出双色球500万》分析

**出版**: 广东经济出版社, 2013年8月, 252页, ISBN 978-7-5454-2575-8

---

## 全书结构

核心内容分两部分:

### 1. 胆拖投注组合数学 (p1-5)
计算不同胆数下全包剩余号码需要的注数。纯组合数学 C(n,k), 无算法。

### 2. 结伴而出统计 (p6-252, 占全书 98%)
33 个号码各自与其他 32 个号码的共现次数表。例如:
- 03 与 21 结伴出现 59 次 (最多)
- 21 与 33 结伴出现 27 次 (最少)

每个号码一页表, 33×32 共现矩阵。无公式、无算法、无验证——纯静态统计数据。
