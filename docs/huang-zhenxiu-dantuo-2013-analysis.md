## 核心内容

全书98%是纯统计数据表。例如：

```
表19 与12结伴而出的情况
01/12: 50次  03/12: 42次  ...
12/16: 33次  12/30: 49次
```

每个号码与其他32个号码的共现次数 → 33个两码表。
再延伸到三码共现表 → 数百个三码表。

**无公式、无算法、无预测模型、无验证方法。**
